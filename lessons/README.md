## Mission Format

Each mission file follows this structure:

1. **Story Setup** - Fictional scenario at CyberShield Corp
2. **Learning Goals** - One coding concept + one security concept
3. **Task Description** - What the player needs to do
4. **Starter Code** - Code to begin with
5. **Solution** - Correct approach
6. **Common Mistakes** - What beginners often do wrong
